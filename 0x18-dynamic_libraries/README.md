Dynamic libraries in the c programming language

A library is a collection of object files grouped into one unit to ease programming work distribution or to structure software development into units. You can create two kinds of C libraries.

Static libraries: this are libraries that are compiled during run-time, they ofer better performance than dynamic libraries, but create larger files

Dynamic libraries: They're not part of the program link with them, rather, they're saved in memory to be used by multiple programs. They offer less performance than static libraries but make for smaller files
